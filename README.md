# とみ歯科クリニック 歯科衛生士 奨学金制度ページ

このリポジトリは、とみ歯科クリニック（武豊町）の奨学金制度ページを Netlify で確認用にホスティングするためのものです。

## ファイル構成

```
.
├── index.html      奨学金ページ本体
├── images/         画像（11枚）
└── netlify.toml    Netlify の設定
```

## 確認URL

GitHub + Netlify 連携により、`main` ブランチに push すると自動的にデプロイされます。

公開URL: https://[Netlifyで設定したサブドメイン].netlify.app/

## 本番設置時の注意

本番の `tomi-dent.com/information/scholarship.html` に設置する際は、
このリポジトリではなく、別途用意した「scholarship_files/scholarship.html」
（相対パス版）を制作会社に渡してください。

このリポジトリは確認用に限定したものです。
